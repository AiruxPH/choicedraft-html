ChoiceDraft

A Web-Based Multiple Choice Test Platform

Version 1.3.0

Our Mission

ChoiceDraft empowers educators and content creators to build, manage, and share high-quality multiple-choice tests with ease. From single quizzes to comprehensive test series, we've got you covered.

Features

User Management

Create and manage accounts for test makers and takers.

Test Management

Create, edit, and organize multiple tests in one place.

Question Pool

Build a pool of questions and shuffle them for each student.

Smart Scoring

Automatic grading with detailed score breakdown.

Report System

Comprehensive performance reports for test series.

Text Formatting

Apply bold, italic, and underline formatting to questions.

Answer Key

Review correct answers with explanations.

Test Builder

Manage test structure, settings, and time limits.

Feedback System

Give personalized feedback to each test taker.

Collaboration

Invite colleagues to co-create tests as editors or viewers.

ChoiceDraft is a live, server-backed platform. Your data is stored securely in a managed database. Please use responsibly and avoid sharing sensitive personal information in test content.
